Angular binding example 1 - Checkbox Example with CheckAll box

Infinity loop could happen in Angular if you bind 1+ UI models with ngChange wich change each other. Usually you can prevent this by reducing and decoupling UI-binded models like above example. Or else you have to rely on the hidden $parse mechanism to evaluate all models unchanged (before allowed iteration runs out), which is tricker and create maintainease issue
